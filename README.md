Overview

A professional QR Code Generator web application built with Python and Streamlit.
Generates customizable QR codes with multiple types, logo embedding, batch generation, and download options in PNG and SVG formats.


Key Features

* Generate QR codes for Text/URL, Wi-Fi and VCard
* Custom QR & background colors
* Logo embedding support
* Batch QR generation from TXT/CSV files
* Download QR codes in PNG & SVG formats
* Interactive and user-friendly UI built with Streamlit

Usage

Open the Live Demo
* in your browser
* Select the QR code type: Text/URL, Wi-Fi, VCard, or Batch
* Customize QR color, background color, and optionally add a logo
* Upload TXT/CSV file if generating batch QR codes
* Click Generate QR to preview codes
* Download QR codes in PNG or SVG format or ZIP for multiple QR codes

Technologies

* Python – Core programming
* Streamlit – Web interface
* QRCode Library – QR code generation
* Pillow (PIL) – Image processing
